Cores em CSS

Este título está colorindo usando cores por nome

Já este está usando cores pelo código Hexadecimal

Este outro está colorindo usando códigos RGB (red, green, blue)

Este está utilizando código HSL(hue, saturation, lightness)

As cores resultantes são exatamente as mesmas, mas o formato de representação muda conforme o padrão escolhido.

Também é possível usar transparência, usando o formato RGBA (red, green, blue, alpha)
